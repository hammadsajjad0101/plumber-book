## 1. Registration & Onboarding

### Step 1.1: Sign Up
```
User Action: Click "Join as Plumber" → Fill registration form

Required Information:
├── Full Name
├── Email Address
├── Phone Number
├── Password
└── Accept Terms & Conditions
```

### Step 1.2: Identity Verification
```
Verification Process:
├── Email verification (click link)
├── Phone verification (SMS code)
├── License verification (upload documents)
└── Background check (optional, for badge)
```

### Step 1.3: Account Activation
```
Once verified:
├── Account becomes active
├── Can access dashboard
└── Profile is NOT visible to customers yet (needs setup)
```

### Step 5.1: New Booking Notification
```
When customer books:
┌─────────────────────────────────────────────────┐
│  NEW BOOKING REQUEST!                           │
│                                                 │
│  Customer: John Smith                           │
│  Phone: (555) 987-6543                          │
│  Service: Leak Repair - $89                     │
│  Date: January 5, 2025                          │
│  Time: 10:00 AM                                 │
│  Address: 456 Oak Street, Apt 2B                │
│                                                 │
│  Notes: "Leak under kitchen sink, dripping      │
│         for 2 days now"                         │
│                                                 │
│  Photos: [3 images attached]                    │
│                                                 │
│  [ACCEPT]  [DECLINE]  [RESCHEDULE]              │
└─────────────────────────────────────────────────┘
```

### Step 5.2: Notification Channels
```
Get notified via:
├── Push Notification (app)
├── SMS Text Message
├── Email
└── In-app Dashboard
```

### Step 5.3: Booking Actions
```
Options:
├── ACCEPT → Confirms booking, customer notified
├── DECLINE → Cancels booking, customer finds another
├── RESCHEDULE → Propose new time
└── MESSAGE → Ask customer a question
```

### Step 5.4: Booking Statuses
```
Status Flow:
PENDING → CONFIRMED → IN PROGRESS → COMPLETED
              ↓
          CANCELLED
```

## 6. Job Execution

### Step 6.1: Day of Appointment
```
Before Leaving:
├── Review job details
├── Check customer notes/photos
├── Prepare tools/materials
└── Get directions to address
```

### Step 6.2: On the Way
```
Notify Customer:
├── "I'm on my way" notification
├── ETA displayed to customer
└── Live location tracking (optional)
```

### Step 6.3: Arrival
```
At Location:
├── Mark "Arrived" in app
├── Meet customer
├── Assess the problem
└── Confirm service needed
```

### Step 6.4: During Job
```
If Additional Work Needed:
├── Discuss with customer
├── Provide quote for extra work
├── Get approval before proceeding
└── Update booking if price changes
```

### Step 6.5: Job Completion
```
Finishing Up:
├── Complete the repair
├── Clean up work area
├── Show customer the fix
├── Collect payment
└── Mark job as "Completed" in app
```

```
Highly recommend!"
```

```
Dashboard:
├── Average Rating: 4.9 ⭐
```

```
Better reviews = Higher ranking:
├── More 5-star reviews → Higher in search
├── Fast response time → "Quick Responder" badge
├── 100+ jobs → "Trusted Pro" badge
└── Verified license → "Verified" badge
```

```
To Get More Bookings:
├── Respond quickly to requests

├── Ask satisfied customers for reviews
└── Expand service area
```
